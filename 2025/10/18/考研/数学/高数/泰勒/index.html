麦克劳林公式与泰勒展开

1. 未定式 $\frac{0}{0}$ 型极限

Taylor 展开：$x \to 0$（对于 $x \to 1$ 的情况，令 $t = x-1$，则 $t \to 0$）

Taylor 公式：

$$f(x) = f(x_0) + \frac{f’(x_0)}{1!}(x-x_0) + \frac{f’’(x_0)}{2!}(x-x_0)^2 + \cdots$$

$$+ \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n + R_n(x)$$

$$R_n(x) = o[(x-x_0)^n]$$

将 $x_0$ 取为 $0$ 即得到麦克劳林展开：

常用麦克劳林展开公式：

指数函数：

$$e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \cdots + \frac{x^n}{n!} + o(x^n)$$

正弦函数：

$$\sin x = \sum_{n=0}^{\infty} \frac{(-1)^n}{(2n+1)!} x^{2n+1} = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots + \frac{(-1)^n}{(2n+1)!} x^{2n+1} + o(x^{2n+1})$$

余弦函数：

$$\cos x = \sum_{n=0}^{\infty} \frac{(-1)^n}{(2n)!} x^{2n} = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots + \frac{(-1)^n}{(2n)!} x^{2n} + o(x^{2n})$$

二项式展开：

$$(1+x)^\alpha = 1 + \sum_{n=1}^{\infty} \frac{\alpha(\alpha-1) \cdots (\alpha-n+1)}{n!} x^n$$

$$= 1 + \alpha x + \frac{\alpha(\alpha-1)}{2!} x^2 + \cdots + \frac{\alpha(\alpha-1) \cdots (\alpha-n+1)}{n!} x^n + o(x^n)$$

$$ \sqrt{1+x} = 1 + \frac{1}{2}x - \frac{1}{8}x^2 + \frac{1}{16}x^3 - \frac{5}{128}x^4 + o(x^4) $$

几何级数：

$$\frac{1}{1+x} = \sum_{n=0}^{\infty} (-1)^n x^n = 1 - x + x^2 - x^3 + \cdots + (-1)^n x^n + o(x^n)$$

$$\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n = 1 + x + x^2 + x^3 + \cdots + x^n + o(x^n)$$

对数函数：

$$\ln(1+x) = \sum_{n=0}^{\infty} \frac{(-1)^n}{n+1} x^{n+1} = x - \frac{1}{2}x^2 + \frac{1}{3}x^3 - \cdots + \frac{(-1)^n}{n+1} x^{n+1} + o(x^{n+1})$$

正切函数：

$$\tan x = x + \frac{1}{3}x^3 + \frac{2}{15}x^5 + o(x^5)$$
